# Arkanoid

## Installation
- Download Release Folder
- Open Arkanoid.exe

## Usage
### Control
- Moving - **LeftArrow, RightArrow**

### General
- Exit - **Esc**
- Restart - **R**
- Pause - **P** or **Space**
- Enable/Disable bot - **A**
- Increase/Decrease ball speed - **+**, **-**
- Increase/Decrease player speed - **<**, **>** (Press **","** and **"."** without Shift)
